# [Pertandingan Battle Code]

A competition I joined in my second semester in Kolej Matrikulasi Melaka. In this competition, I was tasked to solve problems provided by writing a program that could solve the given problems with my coding knowledge.

## 🚀 Flow of the Competition
- There are two rounds, which are the Preliminary Round and Final Round.
- Participants are provided with a problem that asks the participants to write a solution program for each round.
- Participants are competed against others to see which can write a solution program that meets all the criteria the fastest.
- Participants are then placed in rankings based on their performance.

## 🛠️ Conducted With
- **Language:** Java
- **Platform/Website Used:** HackerRank
